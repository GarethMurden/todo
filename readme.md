# ToDo

To do list application inspired by [dooit](https://github.com/dooit-org/dooit), but with tasks grouped by due date instead of workspace.

## Installation

1. Download/clone this repo
2. `pip install -r requirements.txt`

## Usage

Launch with `python todo.py`

### Key bindings
- `left/right arrow keys` - Navigate between date and task lists
- `up/down arrow keys` - Navigate between dates or tasks within each list
- `a` Add task
- `e` Edit task description
- `d` Edit task due date
- `c` Toggle Completed state of task
- `p` Toggle Prioritiy state of task
